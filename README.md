# Two Person Chat

Простой сайт-переписка для двух людей:

- вход без пароля как `Человек 1` или `Человек 2`;
- общий чат в реальном времени;
- минимальный аккуратный интерфейс.

## Локальный запуск

```bash
npm install
npm start
```

После запуска откройте `http://localhost:3000`.

## Деплой на арендованный сервер (Ubuntu)

1. Установить Node.js LTS (например 20.x).
2. Скопировать проект на сервер.
3. В папке проекта выполнить:

```bash
npm install
npm start
```

Для постоянной работы рекомендуется PM2:

```bash
npm install -g pm2
pm2 start server.js --name two-person-chat
pm2 save
pm2 startup
```

Опционально: проксировать через Nginx на домен/HTTPS.

## Важно

История сообщений сейчас хранится в памяти процесса и очищается при перезапуске сервера.  
Если нужно, можно добавить хранение в файл или базу данных.
